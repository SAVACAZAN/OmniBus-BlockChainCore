🔗 RPC API:

Pool Management:
  registerminer(minerData)      - Add a miner dynamically
  minerkeepalive(address)       - Reset 30-second timeout
  getpoolstats()                - Pool status (miners, blocks, rewards)
  getminerstatus()              - Detailed pool + reward info

Blockchain:
  getblockcount()               - Current block height
  getblock(index)               - Get block by height
  getlatestblock()              - Latest block with transactions
  gettransactioncount()         - Total transactions mined
  gettransactionhistory(limit)  - Recent transaction list

Mining:
  getminers()                   - All ACTIVE miners (only!)
  getminerbalances()            - All ACTIVE miner balances

⚙️ TECHNICAL FLOW:

1. REGISTRATION
   Miner → registerminer({id, name, address, hashrate})
   Pool  → Add to minerRegistry Map
   Pool  → Add to activeMinerSet
   Response → { success: true, minerCount: 11 }

2. KEEPALIVE LOOP (every 5 seconds)
   Miner → minerkeepalive(address)
   Pool  → Update lastKeepalive timestamp
   Pool  → Keep in activeMinerSet
   Response → { success: true }

3. MINING (every 2 seconds)
   Pool → Create block #N
   Pool → Count activeMinerSet.size (e.g., 110)
   Pool → Calculate rewardPerMiner = 50 ÷ 110
   Pool → For each miner in activeMinerSet:
          - Create coinbase transaction
          - Update minerBalances[address]
   Pool → Store block with all transactions
   Pool → Increment blockCount

4. CLEANUP (every 5 seconds)
   Pool → Check minerLastBlockTime for all miners
   Pool → If not updated for 30s: remove from activeMinerSet
   Pool → Log: "[POOL] Disconnected: miner-5"
