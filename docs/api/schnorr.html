schnorrSign()

BIP-340 Schnorr sign Signs message with private key using deterministic nonce (RFC 6979 style) Algorithm: 1. d = private_key (scalar) 2. P = d*G (public key point) 3. If P.y is odd, negate d (ensure even Y) 4. t = xor(bytes(d), tagged_hash("BIP0340/aux", a)) 5. rand = tagged_hash("BIP0340/nonce", t || bytes(P) || m) 6. k = int(rand) mod n 7. R = k*G; if R.y is odd, negate k 8. e = int(tagged_hash("BIP0340/challenge", bytes(R) || bytes(P) || m)) mod n 9. sig = bytes(R) || bytes((k + e*d) mod n)
