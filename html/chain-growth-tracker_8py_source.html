3OmniBus BlockChainCore — Chain Growth Tracker

5Monitors the omnibus-chain.dat file size over time and logs growth
6data to MONITORING/data/chain-growth.csv. Calculates the growth rate
7in bytes/hour and raises alerts when growth stops (node not mining)

13 python chain-growth-tracker.py

15 python chain-growth-tracker.py --max-rate 50000000 --stale-minutes 30

39def format_bytes(n: float) -> str:
40 for unit in ("B", "KB", "MB", "GB", "TB"):
41 if abs(n) < 1024:
42 return f"{n:.2f} {unit}"
43 n /= 1024
44 return f"{n:.2f} PB"
