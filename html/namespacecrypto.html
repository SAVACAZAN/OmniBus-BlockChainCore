crypto-edge-cases.py

Test mathematical edge cases in secp256k1-style elliptic-curve cryptography:
  - signing with k = 0, k = n, k = n-1
  - private_key = 0, private_key = n
  - point-at-infinity operations
  - signature malleability: (r, s) -> (r, n-s)
  - weak nonce detection (repeated k leaks private key)
  - low-s normalization check

Outputs: crypto-edge-results.json

repository via git history.  Monitors:

  - BIP-32 derivation path handling changes
  - Crypto-related API surface changes over time

Outputs: crypto-evolution-timeline.json
