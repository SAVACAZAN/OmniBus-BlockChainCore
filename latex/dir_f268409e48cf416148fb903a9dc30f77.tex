\hypertarget{dir_f268409e48cf416148fb903a9dc30f77_autotoc_md10478}{}\doxysubsection{\texorpdfstring{About Tauri}{About Tauri}}\label{dir_f268409e48cf416148fb903a9dc30f77_autotoc_md10478}
Tauri is a polyglot and generic system that is very composable and allows engineers to make a wide variety of applications. It is used for building applications for Desktop Computers using a combination of Rust tools and HTML rendered in a Webview. Apps built with Tauri can ship with any number of pieces of an optional JS API /\+ Rust API so that webviews can control the system via message passing. In fact, developers can extend the default API with their own functionality and bridge the Webview and Rust-\/based backend easily.

Tauri apps can have custom menus and have tray-\/type interfaces. They can be updated, and are managed by the user\textquotesingle{}s operating system as expected. They are very small, because they use the system\textquotesingle{}s webview. They do not ship a runtime, since the final binary is compiled from rust. This makes the reversing of Tauri apps not a trivial task.\hypertarget{dir_f268409e48cf416148fb903a9dc30f77_autotoc_md10479}{}\doxysubsection{\texorpdfstring{This module}{This module}}\label{dir_f268409e48cf416148fb903a9dc30f77_autotoc_md10479}
Written in Typescript and packaged such that it can be used with {\ttfamily npm}, {\ttfamily pnpm}, {\ttfamily yarn}, and {\ttfamily bun}, this library provides a node.\+js runner for common tasks when using Tauri, like {\ttfamily pnpm tauri dev}. For the most part it is a wrapper around \href{https://github.com/tauri-apps/tauri/blob/dev/crates/tauri-cli}{\texttt{tauri-\/cli}}.

To learn more about the details of how all of these pieces fit together, please consult this \href{https://github.com/tauri-apps/tauri/blob/dev/ARCHITECTURE.md}{\texttt{ARCHITECTURE.\+md}} document.\hypertarget{dir_f268409e48cf416148fb903a9dc30f77_autotoc_md10480}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{dir_f268409e48cf416148fb903a9dc30f77_autotoc_md10480}
