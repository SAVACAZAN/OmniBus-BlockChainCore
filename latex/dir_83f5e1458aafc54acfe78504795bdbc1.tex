All of them honour {\ttfamily Ctrl+C} and finish writing their CSV before exiting.\hypertarget{dir_83f5e1458aafc54acfe78504795bdbc1_autotoc_md9835}{}\doxysubsection{\texorpdfstring{Auto-\/restart pattern}{Auto-\/restart pattern}}\label{dir_83f5e1458aafc54acfe78504795bdbc1_autotoc_md9835}
{\ttfamily 01\+\_\+start\+\_\+node.\+ps1} is a forever-\/loop that:\+


\begin{DoxyEnumerate}
\item Spawns the EXE with {\ttfamily Start-\/\+Process -\/Pass\+Thru} and redirected stdout/\+stderr.
\item Calls {\ttfamily Wait\+For\+Exit()}.
\item On any non-\/clean exit, appends to {\ttfamily crashes.\+log} (timestamp + reason + last 50 stderr lines) and sleeps {\ttfamily -\/Restart\+Delay\+Sec} (default 3 s).
\item Exits cleanly when {\ttfamily Ctrl+C} flips {\ttfamily \$KEEP\+\_\+\+RUNNING = \$false}.
\end{DoxyEnumerate}

{\ttfamily 06\+\_\+chaos\+\_\+kill.\+ps1} works in tandem:\+ it picks a random delay between {\ttfamily -\/Min\+Sec} and {\ttfamily -\/Max\+Sec}, then {\ttfamily Get-\/\+Process omnibus-\/node \texorpdfstring{$\vert$}{|} Kill()}. The supervisor immediately restarts. Compare {\ttfamily kills\+\_\+\texorpdfstring{$\ast$}{*}.\doxylink{namespacelog}{log}} against {\ttfamily crashes.\+log} to see whether every kill produced a clean recovery.\hypertarget{dir_83f5e1458aafc54acfe78504795bdbc1_autotoc_md9836}{}\doxysubsection{\texorpdfstring{Graceful shutdown pattern}{Graceful shutdown pattern}}\label{dir_83f5e1458aafc54acfe78504795bdbc1_autotoc_md9836}
Every script:\+


\begin{DoxyItemize}
\item Sets {\ttfamily \$Error\+Action\+Preference = \textquotesingle{}Continue\textquotesingle{}}.
\item Declares {\ttfamily \$script:\+STOP = \$false}.
\item Registers {\ttfamily Console\+Cancel\+Pressed} (or, in {\ttfamily 01\+\_\+start\+\_\+node.\+ps1}, the engine\textquotesingle{}s exit event) to flip {\ttfamily \$STOP = \$true}.
\item Buffers CSV rows in a small {\ttfamily List\+[string]\+} and flushes either every 50 rows or at exit, so a {\ttfamily Ctrl+C} never leaves a half-\/written line.
\item The orchestrator additionally calls {\ttfamily Stop-\/\+Job} /\+ {\ttfamily Remove-\/\+Job} on every background job before exiting.
