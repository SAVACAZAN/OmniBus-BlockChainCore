Creates a new queue with {\ttfamily Promise} apis. It also offers all the methods and properties of the object returned by \doxylink{dir_45fe0d301f9bc4ed8d4d226a3da2225c_fastqueue}{\`{}fastqueue\`{}}{\ttfamily fastqueue} with the modified \doxylink{dir_45fe0d301f9bc4ed8d4d226a3da2225c_pushPromise}{\`{}push\`{}}{\ttfamily push} and \doxylink{dir_45fe0d301f9bc4ed8d4d226a3da2225c_unshiftPromise}{\`{}unshift\`{}}{\ttfamily unshift} methods.

Node v10+ is required to use the promisified version.

Arguments:\+
\begin{DoxyItemize}
\item {\ttfamily that}, optional context of the {\ttfamily worker} function.
\item {\ttfamily worker}, worker function, it would be called with {\ttfamily that} as {\ttfamily this}, if that is specified. It MUST return a {\ttfamily Promise}.
\item {\ttfamily concurrency}, number of concurrent tasks that could be executed in parallel.
\end{DoxyItemize}
