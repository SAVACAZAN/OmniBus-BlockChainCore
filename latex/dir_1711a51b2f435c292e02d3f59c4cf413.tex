Given some data, {\itshape jsesc} returns a stringified representation of that data. jsesc is similar to {\ttfamily JSON.\+stringify()} except:\+


\begin{DoxyEnumerate}
\item it outputs Java\+Script instead of JSON by defaultby default
\end{DoxyEnumerate}, enabling support for data structures like ES6 maps and sets;
\begin{DoxyEnumerate}
\item it offers \doxylink{namespaceapi}{many options}many options
\end{DoxyEnumerate}to customize the output;
\begin{DoxyEnumerate}
\item its output is ASCII-\/safe by defaultby default
\end{DoxyEnumerate}, thanks to its use of \href{https://mathiasbynens.be/notes/javascript-escapes}{\texttt{escape sequences}} where needed.

For any input, jsesc generates the shortest possible valid printable-\/\+ASCII-\/only output. \href{https://mothereff.in/js-escapes}{\texttt{Here’s an online demo.}}

jsesc’s output can be used instead of {\ttfamily JSON.\+stringify}’s to avoid \href{https://en.wikipedia.org/wiki/Mojibake}{\texttt{mojibake}} and other encoding issues, or even to \href{https://twitter.com/annevk/status/380000829643571200}{\texttt{avoid errors}} when passing JSON-\/formatted data (which may contain U+2028 LINE SEPARATOR, U+2029 PARAGRAPH SEPARATOR, or \href{https://esdiscuss.org/topic/code-points-vs-unicode-scalar-values\#content-14}{\texttt{lone surrogates}}) to a Java\+Script parser or an UTF-\/8 encoder.\hypertarget{dir_1711a51b2f435c292e02d3f59c4cf413_autotoc_md7250}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{dir_1711a51b2f435c292e02d3f59c4cf413_autotoc_md7250}

This function takes a value and returns an escaped version of the value where any characters that are not printable ASCII symbols are escaped using the shortest possible (but valid) \href{https://mathiasbynens.be/notes/javascript-escapes}{\texttt{escape sequences for use in Java\+Script strings}}. \DoxyCodeLine{//\ →\ 'Ich\ \(\backslash\)\(\backslash\)u2665\ B\(\backslash\)\(\backslash\)xFCcher'}

\DoxyCodeLine{//\ →\ 'foo\ \(\backslash\)\(\backslash\)uD834\(\backslash\)\(\backslash\)uDF06\ bar'}

The {\ttfamily json} option takes a boolean value ({\ttfamily true} or {\ttfamily false}), and defaults to {\ttfamily false} (disabled). When enabled, the output is valid JSON. \href{https://mathiasbynens.be/notes/javascript-escapes\#hexadecimal}{\texttt{Hexadecimal character escape sequences}} and \href{https://mathiasbynens.be/notes/javascript-escapes\#single}{\texttt{the {\ttfamily \textbackslash{}v} or {\ttfamily \textbackslash{}0} escape sequences}} are not used. To use the {\ttfamily jsesc} binary in your shell, simply install jsesc globally using npm:\+


\begin{DoxyCode}{0}
\DoxyCodeLine{npm\ install\ -\/g\ jsesc}

\end{DoxyCode}


After that you’re able to escape strings from the command line:\+


\begin{DoxyCode}{0}
\DoxyCodeLine{\$\ jsesc\ 'föo\ ♥\ bår\ 𝌆\ baz'}
\DoxyCodeLine{f\(\backslash\)xF6o\ \(\backslash\)u2665\ b\(\backslash\)xE5r\ \(\backslash\)uD834\(\backslash\)uDF06\ baz}

\end{DoxyCode}
