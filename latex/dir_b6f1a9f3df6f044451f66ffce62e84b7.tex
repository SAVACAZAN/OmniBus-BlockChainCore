\hypertarget{dir_b6f1a9f3df6f044451f66ffce62e84b7_autotoc_md5962}{}\doxysubsection{\texorpdfstring{d3-\/ease}{d3-\/ease}}\label{dir_b6f1a9f3df6f044451f66ffce62e84b7_autotoc_md5962}
{\itshape Easing} is a method of distorting time to control apparent motion in animation. It is most commonly used for \href{https://en.wikipedia.org/wiki/12_basic_principles_of_animation\#Slow_In_and_Slow_Out}{\texttt{slow-\/in, slow-\/out}}. By easing time, \href{https://github.com/d3/d3-transition}{\texttt{animated transitions}} are smoother and exhibit more plausible motion.

The easing types in this module implement the ease methodease method, which takes a normalized time {\itshape t} and returns the corresponding “eased” time {\itshape tʹ}. Both the normalized time and the eased time are typically in the range \+[0,1]\+, where 0 represents the start of the animation and 1 represents the end; some easing types, such as \doxylink{dir_b6f1a9f3df6f044451f66ffce62e84b7_easeElastic}{elastic}elastic, may return eased times slightly outside this range. A good easing type should return 0 if {\itshape t} = 0 and 1 if {\itshape t} = 1. See the \href{https://observablehq.com/@d3/easing}{\texttt{easing explorer}} for a visual demonstration.

These easing types are largely based on work by \href{http://robertpenner.com/easing/}{\texttt{Robert Penner}}.\hypertarget{dir_b6f1a9f3df6f044451f66ffce62e84b7_autotoc_md5963}{}\doxysubsection{\texorpdfstring{Installing}{Installing}}\label{dir_b6f1a9f3df6f044451f66ffce62e84b7_autotoc_md5963}

\# {\itshape ease}({\itshape t})

Given the specified normalized time {\itshape t}, typically in the range \+[0,1]\+, returns the “eased” time {\itshape tʹ}, also typically in \+[0,1]\+. 0 represents the start of the animation and 1 represents the end. A good implementation returns 0 if {\itshape t} = 0 and 1 if {\itshape t} = 1. See the \href{https://observablehq.com/@d3/easing}{\texttt{easing explorer}} for a visual demonstration. For example, to apply \doxylink{dir_b6f1a9f3df6f044451f66ffce62e84b7_easeCubic}{cubic}cubic easing:\+

\DoxyCodeLine{const\ te\ =\ d3.easeCubic(t);}

\end{DoxyCode}
