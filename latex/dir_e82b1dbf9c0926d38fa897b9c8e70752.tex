The full happy path for a trading bot is:\+


\begin{DoxyEnumerate}
\item {\bfseries{Connect}} — open RPC + WS connections.
\item {\bfseries{Authenticate}} — derive keypair from mnemonic (BIP-\/32/\+39).
\item {\bfseries{Fund}} — claim faucet on testnet, or receive on mainnet.
\item {\bfseries{Mine (optional)}} — run a miner if you want OMNI directly.
\item {\bfseries{Place orders}} — sign EXCHANGE\+\_\+\+PLACE\+\_\+\+V1 payload, POST Add\+Order.
\item {\bfseries{Listen}} — subscribe to WS for fill events.
\item {\bfseries{Withdraw /\+ send}} — sign a normal TX, POST sendrawtransaction.
\end{DoxyEnumerate}
