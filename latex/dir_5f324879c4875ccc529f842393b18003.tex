This library allows to add custom require hooks, which do not interfere with other require hooks.

This library only works with common\+JS.\hypertarget{dir_5f324879c4875ccc529f842393b18003_autotoc_md7698}{}\doxysubsection{\texorpdfstring{Why?}{Why?}}\label{dir_5f324879c4875ccc529f842393b18003_autotoc_md7698}
Two reasons:\+
\begin{DoxyEnumerate}
\item Babel and istanbul were breaking each other.
\item Everyone seemed to re-\/invent the wheel on this, and everyone wanted a solution that was DRY, simple, easy to use, and made everything Just Work™, while allowing multiple require hooks, in a fashion similar to calling {\ttfamily super}.
\end{DoxyEnumerate}

For some context, see \href{https://github.com/babel/babel/pull/3062}{\texttt{the Babel issue thread}} which started this all, then \href{https://github.com/bcoe/nyc/issues/70}{\texttt{the nyc issue thread}}, where discussion was moved (as we began to discuss just using the code nyc had developed), and finally to \href{https://github.com/danez/pirates/issues/1}{\texttt{\#1}} where discussion was finally moved.\hypertarget{dir_5f324879c4875ccc529f842393b18003_autotoc_md7699}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{dir_5f324879c4875ccc529f842393b18003_autotoc_md7699}

Add a require hook. {\ttfamily hook} must be a function that takes {\ttfamily (code, filename)}, and returns the modified code. {\ttfamily opts} is an optional options object. Available options are:\+ {\ttfamily matcher}, which is a function that accepts a filename, and returns a truthy value if the file should be hooked (defaults to a function that always returns true), falsey if otherwise; {\ttfamily exts}, which is an array of extensions to hook, they should begin with {\ttfamily .} (defaults to {\ttfamily \+[\textquotesingle{}.js\textquotesingle{}]\+}); {\ttfamily ignore\+Node\+Modules}, if true, any file in a {\ttfamily node\+\_\+modules} folder wont be hooked (the matcher also wont be called), if false, then the matcher will be called for any files in {\ttfamily node\+\_\+modules} (defaults to true).\hypertarget{dir_5f324879c4875ccc529f842393b18003_autotoc_md7703}{}\doxysubsection{\texorpdfstring{Projects that use Pirates}{Projects that use Pirates}}\label{dir_5f324879c4875ccc529f842393b18003_autotoc_md7703}
