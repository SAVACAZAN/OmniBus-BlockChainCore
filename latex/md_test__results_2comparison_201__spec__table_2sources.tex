\begin{DoxyItemize}
\item {\bfseries{TPS realistic}} uses sustained on-\/chain non-\/vote/\+non-\/internal throughput where measurable. For Solana this means True\+TPS /\+ non-\/vote tx per second.
\item {\bfseries{Finality}} is reported as the practical wait time before a transaction is irreversibly confirmed by major exchanges/\+applications, not just probabilistic.
\item {\bfseries{TVL}} values from Defi\+Llama-\/style aggregators where De\+Fi exists; "{}0"{} for chains without smart-\/contract De\+Fi.
\item {\bfseries{Mainnet date}} is the date of public mainnet token genesis, not testnet/\+devnet.
\item {\bfseries{Where ambiguous}}, the lower bound is reported, with notes flagging stress-\/test peaks vs. sustained values.
\end{DoxyItemize}
