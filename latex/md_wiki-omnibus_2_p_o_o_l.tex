The Omni\+Bus Mining Pool is a {\bfseries{no-\/hardcode, dynamic registration system}} where:\+
\begin{DoxyItemize}
\item ✅ Pool starts with {\bfseries{zero miners}}
\item ✅ Miners {\bfseries{register themselves}} at runtime
\item ✅ Rewards {\bfseries{distributed equally}} to all active miners
\item ✅ Miners {\bfseries{join/\+leave dynamically}} without pool restart
\item ✅ Each miner has its {\bfseries{own BIP-\/39 wallet}} with deterministic addresses
\end{DoxyItemize}

\hypertarget{md_wiki-omnibus_2_p_o_o_l_autotoc_md11750}{}\doxysubsection{\texorpdfstring{Block Reward:\+ 50 OMNI per block}{Block Reward:\+ 50 OMNI per block}}\label{md_wiki-omnibus_2_p_o_o_l_autotoc_md11750}
When a block is mined:\+


\begin{DoxyEnumerate}
\item {\bfseries{Count active miners}} (those that sent keepalive in last 30s)
\begin{DoxyItemize}
\item Example:\+ 110 active miners
\end{DoxyItemize}
\item {\bfseries{Divide reward equally}}
\begin{DoxyItemize}
\item Block reward = 50 OMNI = 5,000,000,000 SAT
\item Per-\/miner reward = 50 OMNI ÷ 110 = 0.\+4545 OMNI per miner
\item Per-\/miner SAT = 5,000,000,000 ÷ 110 = 45,454,545 SAT per miner
\end{DoxyItemize}
\item {\bfseries{Create coinbase transactions}} (one per active miner)
\begin{DoxyItemize}
\item Block \#100 → 110 transactions (one for each active miner)
\item Total transactions that block = 110
\end{DoxyItemize}
\item {\bfseries{Update miner balances}}
\begin{DoxyItemize}
\item Each miner\textquotesingle{}s balance += 45,454,545 SAT
\item Total pool balance += 50 OMNI
\end{DoxyItemize}
\end{DoxyEnumerate}\hypertarget{md_wiki-omnibus_2_p_o_o_l_autotoc_md11751}{}\doxysubsection{\texorpdfstring{Fair Distribution}{Fair Distribution}}\label{md_wiki-omnibus_2_p_o_o_l_autotoc_md11751}
✅ If 10 miners → each gets 5 OMNI per block ✅ If 110 miners → each gets 0.\+454 OMNI per block ✅ If miner disconnects (no keepalive) → removed from active set ✅ If 5 miners remain → each gets 10 OMNI per block

{\bfseries{Result}}:\+ {\bfseries{Perfectly equal}} reward distribution, scales automatically.
