-\/fuzzer 

\begin{DoxyVerb}OmniBus BlockChainCore — Differential Fuzzer for secp256k1 / SHA-256 / RIPEMD-160.

Generates random private keys and messages, signs/hashes with our Zig node
(via RPC or subprocess) and cross-checks with Python hashlib + optional
coincurve for secp256k1 agreement.  Reports any byte-level divergences.
\end{DoxyVerb}
