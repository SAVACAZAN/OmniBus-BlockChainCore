Quick Start

# Clone & build
$ git clone https://github.com/SAVACAZAN/OmniBus-BlockChainCore.git
$ cd OmniBus-BlockChainCore
$ zig build

# Start seed node
$ ./zig-out/bin/omnibus-node --mode seed --node-id node-1 --port 9000
[INFO] OmniBus Node v0.1.0 starting...
[INFO] Genesis block initialized: 0000000000...
[INFO] RPC server listening on :8332
[INFO] WebSocket server on :8334
[INFO] P2P listening on :9000

# Start miner (new terminal)
$ ./zig-out/bin/omnibus-node --mode miner --node-id miner-1 --seed-host 127.0.0.1 --seed-port 9000
[INFO] Connected to seed node 127.0.0.1:9000
[INFO] Mining started... difficulty=4

# Docker (alternative)
$ docker-compose up -d
